Project Title: Steam Market Analysis & SDG 9 Dashboard

Project Overview

This project analyzes a dataset of 27,000+ Steam games to explore market trends, pricing strategies, and industry growth. The goal is to demonstrate how digital platforms act as essential infrastructure for global innovation, aligning with UN Sustainable Development Goal 9 (Industry, Innovation, and Infrastructure).

Key Features

3-Page Interactive Dashboard: Includes a Market Overview, Deep-Dive Analysis, and SDG 9 Impact report.
Star Schema Architecture: Optimized data model with a central Fact table and 4 Dimension tables for high performance.
Predictive Analytics: Features a "What-If" parameter to forecast revenue based on price fluctuations.
Aesthetic Design: Implemented a "Purple Glassmorphism" UI for a modern, professional look.

Technical Stack -

Tool: Power BI Desktop / Power BI Service
Data Modeling: Star Schema logic
DAX Measures: Positive Rating %, Total Revenue, Projected Revenue, etc.
UI/UX: Glassmorphism, Synced Slicers, and Page Navigation.

SDG 9 Mapping -

Infrastructure: Tracks the growth of 26k+ games as a digital backbone.
Innovation: Analyzes how Indie developers (small enterprises) access global markets.
Inclusivity: Measures technical reach across Windows, Mac, and Linux.
